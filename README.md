## ✨ Key Features

- **Pixel-Perfect UI:** Precise implementation of provided design specifications with a focus on typography and spacing.
- **Vanilla Implementation:** Built entirely with standard web technologies (HTML5, CSS3, and JavaScript) for maximum performance and zero build-step overhead.
- **Responsive Design:** Optimized layout that adapts seamlessly across mobile, tablet, and desktop viewports.
- **Smooth Interactions:** Custom CSS transitions and JavaScript event handling for an interactive user experience.
- **Clean Codebase:** Organized structure following best practices for naming conventions and document flow.
